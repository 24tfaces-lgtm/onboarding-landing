Инструкция по установке и входу в CRM

Пошаговое руководство для Windows: VPN (OpenConnect) → вход в CRM через Telegram

Начать установку

Следуй шагам ниже. Если что‑то не получается — обратись к своему РГП.

  1. 1. Скачай OpenConnect

    Нажми на кнопку ниже и скачай установщик OpenConnect для Windows.

    Скачать OpenConnect

  2. 2. Установи OpenConnect

    Запусти скачанный файл .exe и следуй инструкциям мастера установки. По завершении запусти программу.

  3. 3. Подключись к VPN

    Открой OpenConnect, введи адрес сервера (даёт РГП) → нажми Connect → дождись статуса «Connected».

  4. 4. Перейди на сайт CRM

    Открой CRM по ссылке: crm.adtika.com

  5. 5. Авторизуйся через Telegram

    Выбери вход через Telegram → укажи номер телефона → введи код из Telegram → подтверди вход.

  6. 6. Проверь звук и микрофон

    В настройках CRM выбери правильный микрофон/гарнитуру и сделай тестовый звонок.

  7. 7. Если не получается

    • Не приходит код в Telegram — проверь Telegram Desktop и интернет, запроси код повторно.
    • VPN не подключается — перепроверь адрес сервера и учётные данные, перезапусти OpenConnect.
    • Обратись к своему РГП за доступами и проверкой настроек.

Полезно знать

  • Для стабильной работы рекомендуем проводной интернет или Wi‑Fi 5 ГГц.
  • Не забывай обновлять Windows и драйверы гарнитуры.
  • При любых сбоях сначала отключи и заново подключи VPN.
